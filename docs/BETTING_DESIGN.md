# 🎲 竞猜赔率与下注选项设计方案

## 📊 基础赔率设计原则

### 赔率计算公式
```
赔率 = 1 / 预期概率 × 庄家系数(0.9)

示例：
- 预期概率 50% → 赔率 = 1/0.5 × 0.9 = 1.8
- 预期概率 80% → 赔率 = 1/0.8 × 0.9 = 1.125 ≈ 1.15
- 预期概率 20% → 赔率 = 1/0.2 × 0.9 = 4.5
```

### 庄家系数说明
- 0.9 的庄家系数保证长期平衡
- 所有选项赔率的倒数之和应 > 1（庄家优势）
- 实际可根据现场气氛微调

## 💡 现场运营建议

### 赔率动态调整
1. **冷门选项加成**：如果某选项无人下注，可临时提高赔率吸引
2. **热门选项控制**：如果某选项下注过多，可降低赔率控制风险
3. **玩家表现参考**：根据玩家前几关表现，调整后续关卡赔率

### 竞猜开启时机
```
对战准备阶段 → 开放所有竞猜（2分钟）
玩家确认Build → 锁定胜负盘，其他可继续
对战开始 → 锁定所有竞猜
对战结束 → 逐一结算
```

### 娱乐性增强
- **连胜奖励**：连续猜对3次积分 ×1.5
- **翻盘奖励**：积分最后一名猜对特殊选项翻倍
- **预言家称号**：单场全中者获得称号展示
